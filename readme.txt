== Description ==

### Add iPaymu gateway to WooCommerce ###

WooCommerce Ipaymu Payment Gateway (iPaymu for WooCommerce) is a non-official iPaymu payment gateway for Wordpress WooCommerce store.

Please notice that WooCommerce must be installed and activated.

The iPaymu for WooCommerce plugin was developed without any incentive from [iPaymu]. 
None of the plugin developer has ties to this company.

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Create an account on [iPaymu] please use my aff link ^__^ (https://my.ipaymu.com?rid=masedi);
* Navigate to WooCommerce -> Settings -> Payment Gateways, choose iPaymu Checkout and fill in your Username and API Key.
